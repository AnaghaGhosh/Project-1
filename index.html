Schedule Manager

A Procrastinator's Path to Consistency

The Schedule Manager is a user-friendly and easy to operate interface. It has been created using Python and stores data in Structured Query Language. The program is efficient at managing your day, or even saving your plan for the week. You can enter details regarding your lectures, timings, tasks to be completed with short breaks and more. You will be able to mark them as complete and the amount of work remaining can be tracked at any point in time. The percentage progress is also displayed. Editing all these details, as many times as you require, is quite simple. There are features to search and display information from your schedule. To sum it up, this Schedule Manager will do what is necessary to help you plan.

Data Structure:

Task Info: Day- String, Time Limit- Numeric, S.No- Numeric, Duration- String, Task Name- String, Status- String

Storage (In SQL Tables): S.No- Integer, Duration- Char, Task Name- Varchar, Status- Char

Progress Info: Total Tasks- Numeric, No. of Tasks Completed- Numeric, No. of Tasks Left: Numeric, Percent Completed: Numeric
